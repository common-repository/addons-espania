Core functionality for Espania Theme

== Description ==

This plugin provides additional functionality for the Espania WordPress theme.

= Main Features =

* **Custom taxonomies** - for theme portfolio
* **Shortcodes** - set of shortcodes that will cover all your needs
* **Shortcodes generator** for for easier use
* **Widgets** - set of unique widgets

== Installation ==

1. Install and activate Espania theme
2. Upload addons-espania to the /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= I activate plugin but nothing is happen =

This plugin works only with Espania WordPress theme. Make sure to install and activate it first.
